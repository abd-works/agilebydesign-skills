# Rule: Domain terms appear as control labels verbatim

When a control label comes from the domain terms file, copy it character-for-character — same casing, same spacing, same wording.

**DO** use the exact string from the domain terms file as the control label.

**DO NOT** paraphrase, abbreviate, capitalise differently, or split compound terms.

**Example (pass):** Domain terms file defines `COH game directory` — control label is `COH game directory`.

**Example (fail):** Label shows `Game folder`, `COH path`, `City of Heroes folder`, or `COH dir`.

**Example (fail):** Domain terms file writes `crowd manager` (lowercase) — label shows `Crowd Manager` or `CrowdManager`.
