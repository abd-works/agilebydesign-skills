# Context to memory package

Convert office documents to markdown, chunk for retrieval, embed vectors, and search memory.

**Skills:** abd-convert-to-markdown, abd-chunk-markdown, abd-embed-vectors, abd-search-memory

## Deploy

```bash
python scripts/deploy_family_package.py --package context-to-memory --to <workspace>
```

Or deploy all families:

```powershell
& scripts/deploy-skills.ps1 -Force
```
