# built/

Pre-generated phase slices assembled by `python scripts/build.py` (static delivery mode only).

**Do not hand-edit files in this folder.** They are regenerated every time `build.py` runs. Edit `content/parts/phases/` and `content/parts/library/` instead, then rebuild.

In `runtime_injection` mode this folder is unused — `generate.py` assembles context on the fly.
