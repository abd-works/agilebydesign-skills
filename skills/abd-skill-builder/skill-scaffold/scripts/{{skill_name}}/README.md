# scripts/{{skill_name}}/

Rename this folder to your skill name (e.g. `scripts/abd-my-skill/`).

Place all scripts that are specific to this skill here. Examples:
- data extraction or transformation scripts
- phase-specific code that drives a build or generation step
- anything that calls external tools or APIs for this skill

Universal scripts (build, generate, run_scanners, etc.) live in `scripts/base/` and should not be modified.
