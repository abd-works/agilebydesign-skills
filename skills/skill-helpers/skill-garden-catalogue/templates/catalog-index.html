AGENT

A role-based AI collaborator

Has a defined persona, goals, workflow & checkpoint protocol. Examples: abd-delivery-lead (orchestrator), abd-team-member (Product Owner / Analyst / Engineer).

SKILL

A reusable domain expertise module

SKILL.md + rules + scanners + templates + scripts. Claude auto-discovers and loads it when triggered. Example: abd-story-mapping, abd-acceptance-criteria.

PLUGIN

A deployable bundle

Groups agents + skills + slash commands + hooks into one installable package. Deploy to your whole team via managed settings in VSCode or Claude Code.

TOOL

A specific capability granted to an agent

Bash execution, file ops, web search, MCP server calls. Tools let agents DO things — not just know things. Every agent has a specific tool set.
