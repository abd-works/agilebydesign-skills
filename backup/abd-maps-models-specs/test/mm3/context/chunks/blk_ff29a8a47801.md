INCREASED ACTION

-1 TO -3 COST PER RANK

Using an effect requires one of the following types of ac-
tions:  standard,  move,  free,  or  reaction.  Each  increase  in
the  required  action  from  that  effect’s  base  action  type
(free to move, for example) is a –1 cost per rank modifier.
