COUNTERS CONCEALMENT

2 RANKS

A sense type with this trait ignores the Concealment effect
of a particular descriptor; you sense the subject of the ef-
fect normally, as if the Concealment wasn’t even there. So
if  you  have  vision  that  Counters  Invisibility,  for  example,
