DROPPING OBJECTS

Simply  dropping  a  created  object  on  a  target  is  treated
like an Area Effect attack based on the object’s size (see
the Area extra in the Powers chapter). The object inflicts
damage equal to its Toughness, and targets get a Dodge
check to evade the falling object. A successful check re-
sults in no damage (rather than the usual half damage).

While  a  created  object  can  potentially  be  wielded  as  an
improvised  weapon,  the  effect  cannot  otherwise  create
attacks  or  other  effects;  you  must  acquire  these  effects
separately (perhaps as Alternate Effects).
