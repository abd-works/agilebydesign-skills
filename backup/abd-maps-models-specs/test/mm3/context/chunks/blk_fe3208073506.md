APPLYING MODIFIERS

An extra increases an effect’s cost per rank by a set amount
(usually  1  point)  while  a  flaw  decreases  the  effect’s  cost
per rank by a set amount (usually 1 point as well). To de-
termine the effect’s final cost per rank, take the base cost,
add up all the extras, and subtract all of the flaws.

Modified Cost = base effect cost +
extras - flaws
