SENSES

SENSORY

Action: None • Range: Personal
Duration: Permanent • Cost: 1 point per rank

One or more of your senses are improved, or you have ad-
ditional  sensory  abilities  beyond  the  normal  five  senses.
Allocate ranks in Senses to the following effects. Some op-
tions require more than one rank, noted in their descrip-
tions. So if you have Senses 5, for example, you can have
darkvision  (2  ranks),  direction  sense  (1  rank),  distance

sense  (1  rank),  and  ultra-hearing  (1  rank),  or  any  other
combination adding up to 5 ranks.

Like  all  sensory  effects,  Senses  uses  the  sense  types  de-
scribed on page 178 as descriptors.
