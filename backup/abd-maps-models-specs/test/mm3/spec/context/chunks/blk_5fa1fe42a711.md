FEATURES

A  headquarters  may  have  a  number  of  features,  chosen
from the list below. A headquarters automatically has the
basic structural amenities like doors and windows, power
outlets, utilities, and so forth at no cost. Each feature costs
1  equipment  point.  You  can  invent  additional  features,
basing them off the ones given here, with the permission
of the Gamemaster.
