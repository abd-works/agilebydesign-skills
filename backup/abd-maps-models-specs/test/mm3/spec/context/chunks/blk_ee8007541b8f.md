Also, because the measurements on the table operate at
the  same  scale  (roughly  doubling  every  rank),  it  is  pos-
sible to use it to quickly figure out relationships between
things like mass, distance, speed, and time:

Distance Rank = Time Rank + Speed Rank

To determine the distance a hero covers in a given amount
of time, add the rank of the time to the rank of the hero’s

speed,  with  normal  human  ground  speed  being  rank  0.
So a normal person can cover 2 miles in an hour (time 9 +
speed 0 = 9, the rank for 2 miles). In fact, with normal hu-
man speeds, you can just directly compare the time and
distance columns of the table! As another example, a hero
with Flight 12 can cover 8,000 miles in an hour! That’s 12
(speed) + 9 (time) = 21, the rank for 8,000 miles. The same
character  can  go  an  amazing  16  miles  in  just  6  seconds
(the time of one action round)!

Time Rank = Distance Rank - Speed Rank

Reversing  the  previous  formula,  we  can  also  figure  out
how long it takes someone at a particular speed to cov-
er  a  given  distance,  by  subtracting  the  speed  rank  from
the distance rank to get a time rank. So a normal human
(speed  0)  walking  30  miles  (distance  13)  takes  about  16
hours. A hero with Speed 14 covers the same distance in
(13 – 14 = –1) just 3 seconds!

Throwing Distance Rank =
Strength Rank - Mass Rank

As  another  example,  the  distance  rank  a  hero  can  throw
something equals the hero’s Strength rank minus the mass
rank of the object. So a hero with Strength 10 (able to lift 25
tons), picks up a 10-ton truck (mass rank 9). Since 10-9=1,
the hero can then toss the truck rank 1 distance (60 feet)!
