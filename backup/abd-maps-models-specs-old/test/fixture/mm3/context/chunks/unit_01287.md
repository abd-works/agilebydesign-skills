## DC

5 10 15 20 25 Note that the Vehicles skill does not cover riding animal mounts. For that, use the Expertise: Riding skill, based on Agility, with the same guidelines as given for Vehicles skill checks. At the Gamemaster’s discretion, skills like Athletics can serve for riding mounts (perhaps with a circumstance penalty), especially if riding is a fairly uncommon skill, as it is in the modern world.
