## FIRE PREVENTION SYSTEM

The headquarters is equipped with an automatic system for detecting and extinguishing fires. Any large open flame sets the system off (beware, fire-using heroes!). It functions as a Nullify Fire 5 effect. A computer-controlled fire prevention system can be programmed to ignore cer- tain sources of fire or the system can be placed on manual control (requiring someone to throw a switch in order to activate it).
