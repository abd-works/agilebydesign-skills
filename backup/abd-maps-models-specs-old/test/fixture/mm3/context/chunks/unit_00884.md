## SKILL BASICS

Skills are learned abilities, a combination of training (the skill) and natural talent (an ability rank). Each skill has a rank, used as a bonus to the die roll when using the skill. To make a skill check, roll: d20 + skill rank + ability modifier + miscellaneous modifiers
